# Public Portfolio

This folder contains a public, read-only portfolio page.

## What it includes
- A landing page that can be shared publicly
- No wallet connection
- No admin routes
- No private dashboard or hidden actions

## How to publish
Deploy the contents of this folder to any static host:
- GitHub Pages
- Vercel
- Netlify

If you use GitHub Pages, point the site to the `public-portfolio` folder contents or move `index.html` to the repository root for the published branch.

## Before sharing
Replace the placeholder contact links in `index.html` with your real email, LinkedIn, and GitHub URLs.
